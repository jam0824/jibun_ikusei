# 自分育成 - ブラウジングトラッカー (Chrome拡張)

ブラウザの閲覧履歴を成長ログに変換する Chrome 拡張です。AIがページ内容を分類し、学習・仕事などの成長コンテンツにはXPを付与、ブロックリスト上の娯楽サイトを長時間閲覧するとペナルティが発生します。

## 機能

- **AI分類**: OpenAI / Gemini がページを8カテゴリに自動分類
- **XP付与**: 成長コンテンツを30分閲覧すると +2 XP、以降60分ごとに追加 +2 XP
- **ペナルティ**: ブロックリストのサイトを60分閲覧すると -5 XP（50分で事前警告）
- **Lily通知**: 閾値達成時にトースト通知で報告
- **ポップアップ**: 今日の閲覧進捗をリアルタイム表示
- **週間レポート**: 週次の成長サマリーを自動生成

## インストール・デプロイ手順

### 1. 依存パッケージのインストール

```bash
cd chrome-extension
npm install
```

### 2. ビルド

```bash
npm run build
```

`dist/` フォルダにビルド済みファイルが生成されます。

### 3. Chrome に読み込む

1. Chrome で `chrome://extensions` を開く
2. 右上の **「デベロッパーモード」** をオンにする
3. **「パッケージ化されていない拡張機能を読み込む」** をクリック
4. `chrome-extension/dist` フォルダを選択

拡張がインストールされ、ツールバーにアイコンが表示されます。

### 4. 拡張を更新する（コード変更後）

```bash
npm run build
```

ビルド後、`chrome://extensions` で拡張の **更新ボタン（↺）** をクリックする。

## 初期設定

拡張アイコンを右クリック → **「オプション」** を開く。

### AIプロバイダー設定

| 項目 | 説明 |
|------|------|
| OpenAI APIキー | `sk-...` で始まるキー。GPT-5.4 を使用 |
| Gemini APIキー | Google AI Studio で取得したキー。gemini-2.5-flash を使用 |

どちらか一方を設定し、「保存」→「接続テスト」で動作確認できます。

### サーバー接続設定

| 項目 | 説明 |
|------|------|
| サーバーURL | 自分育成アプリのAPIベースURL（例: `https://xxx.execute-api.ap-northeast-1.amazonaws.com`） |

「URL保存」をクリックして保存してください。

### ログイン

自分育成アプリと同じメールアドレス・パスワードでログインします。ログイン後は「ログイン中: メールアドレス」と表示されます。

### ブロックリスト

ペナルティを適用したいドメインを登録します。

- 例: `youtube.com`、`twitter.com`、`instagram.com`
- ドメインを入力して「追加」または Enter キーで登録
- 登録済みドメインを60分以上閲覧すると -5 XP

## 使い方

### ポップアップ

ツールバーのアイコンをクリックすると、今日の閲覧進捗が表示されます。

```
今日の進捗
  成長 2h 30m
  その他 45m
  注意 15m

  クエスト達成: 3回 / ペナルティ: 0回

ドメイン別
  github.com     - 45分
  qiita.com      - 30分
  ...
```

### 通知（Lilyトースト）

閾値に達すると画面右下にトーストが表示されます。

| タイミング | メッセージ |
|-----------|-----------|
| 成長30分達成 | 「学習クエスト達成。+2 XP 獲得しました」 |
| 成長90分達成 | 「集中が続いていますね。さらに +2 XP です」 |
| ブロック50分 | 「もうすぐ1時間です。少し休憩してみては？」 |
| ブロック60分 | 「少し長く滞在しすぎました。-5 XP」 |

## ストレージ

| データ | 場所 | タイミング |
|--------|------|-----------|
| APIキー・設定 | `chrome.storage.local` | ブラウザ再起動後も保持 |
| ログイン情報 (Cognito JWT) | `chrome.storage.local` | ブラウザ再起動後も保持 |
| 今日の閲覧データ | `chrome.storage.local` | 日付変更時に自動リセット |
| 分類キャッシュ | `chrome.storage.local` | 30日間保持 |

## トラブルシューティング

**ポップアップに「まだデータがありません」と表示される**
- 少しブラウジングすると計測が始まります。サービスワーカーが30秒ごとにデータを保存します。

**APIキーを設定したのに「接続テスト」が失敗する**
- APIキーが正しいか確認してください。
- OpenAI: `sk-proj-...` または `sk-...` で始まるキー
- Gemini: Google AI Studio (`aistudio.google.com`) で生成したキー

**XPがサーバーに反映されない**
- オプションページでサーバーURLとログインが正しく設定されているか確認してください。
- オフライン時はキューに蓄積され、次回オンライン時に自動同期されます。

**拡張を更新しても設定が消えない**
- 設定は `chrome.storage.local` に保存されるため、拡張の更新では消えません。アンインストール時のみ消えます。
