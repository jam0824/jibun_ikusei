# 週次ふりかえり画面遷移補足 v0.1

更新日: 2026-04-11

```text
[ホーム画面]
  └─ 未読の先週ふりかえりカード -> [週次ふりかえり画面]

[記録画面(filter=week, tab=quests)]
  └─ 先週のふりかえりを見る -> [週次ふりかえり画面]

[週次ふりかえり画面]
  ├─ 記録へ -> [記録画面(filter=week)]
  └─ ボトムナビ -> [ホーム画面] / [クエスト一覧] / [スキル画面] / [記録画面]
```

- 週次ふりかえり画面は `先週` のみを扱う。
- 初回表示時にその週の Lily コメントと来週のおすすめを生成し、以後はキャッシュを再利用する。
