# 週次ふりかえり画面遷移補足 v0.1

更新日: 2026-04-18

## 1. 正式 route

- 週次ふりかえり画面の canonical route は `/records/review/weekly` とする。
- 旧 `/weekly-reflection` は redirect-only alias とし、画面の正式入口としては扱わない。

## 2. 導線

```text
[ホーム]
  └─ 未読の先週ふりかえりカード → [/records/review/weekly]

[成長]
  └─ 週次ふりかえりを見る → [/records/review/weekly]

[成長記録]
  └─ 先週のふりかえりを見る → [/records/review/weekly]

[/records/review/weekly]
  ├─ 記録へ → [/records/growth?range=week]
  └─ 下部ナビ → [ホーム] / [クエスト] / [成長] / [記録]
```

## 3. 役割

- 週次ふりかえり画面は `先週` のみを扱う。
- 初回表示時にその週の Lily コメントと来週のおすすめを生成し、以後はキャッシュを再利用する。
- ホーム側では「未読の先週ふりかえりがあるか」を示し、詳細の正式な置き場は週次ふりかえり画面に一本化する。

## 4. 実装メモ

- ホーム、成長、成長記録のいずれから来ても同じ route に着地させる。
- 「記録へ」は記録ハブではなく `/records/growth?range=week` に返し、直前の文脈を維持する。
